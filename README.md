# Scalera Solutions

Next.js website for Scalera Solutions — AI automation for client acquisition.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to DigitalOcean App Platform

1. Push this repo to GitHub
2. Go to DigitalOcean → App Platform → Create App
3. Select this GitHub repo
4. Select branch: `main`
5. DigitalOcean will auto-detect Next.js
6. Click Deploy

## Build command
```
npm run build
```

## Start command
```
npm run start
```

## Environment
- Node.js 18+
- Next.js 14
- Tailwind CSS
